## Problem Statement
Rising land surface temperatures present an unpredictable challenge to agricultural planning in East Africa. This project models historical temperature changes over time to provide clear statistical trends for modern planting applications.

## Data Source
The raw data for this analysis was mined directly from the FAOSTAT "Temperature change on land" module, utilizing 10 separate chronological CSV sequences for Kenya.

## Methodology
* *Data Cleansing:* Employed Python's pandas library to programmatically merge 10 structural records and drop missing data lines.
* *Scientific Computing:* Used the scipy.stats library to compute a line of best fit and quantify linear climate acceleration.
